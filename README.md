# Among Us MuteAllBot
Bot to mute everybody in a voice channel. 


## Setup

```
yarn 
export DISCORD_BOT_TOKEN=<use your discord bot token here>
```

## Run locally

```
yarn start
```

## Commands

`/mute`, `/ssh`, `/m` to mute server
`/unmute`, `/speak`, `/u` to unmute server
